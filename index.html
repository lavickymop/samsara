Info Samsara

Samsara est une application qui suit le cycle menstruel de la femme dans le but de connaître ses périodes de fécondité.

Samsara n’oublie jamais!

Samsara analyse le rapport entre la variation de température de votre corps et vos périodes de menstruation, ce qui permettra d’établir le modèle de votre cycle menstruel.

Pour que l’application fonctionne vous devez simplement répondre à deux questions à chaque matin.

Mode d’emploi
  • Créer un compte Samsara
  • Répondre aux deux questions à tous les matins
  • Consulter la visualisation des données (peut prendre de 3 à 5 semaines selon la régularité de vos règles)
  • Samsara a besoin de 3 à 5 semaines de données pour comprendre votre cycle. Vous n'aurez donc pas la visualisation complète lors des premières utilisations.
  • Si votre cycle menstruel est extrêmement irrégulier, la compréhension de ce cycle pourrait prendre plus de temps.
